Installation

Get LunaOS up and running in minutes with our simple installation process. 

# Clone the repository
git clone https://github.com/lunaos-ai/lunaos.git
cd lunaos

# Install dependencies
make install

# Start LunaOS
make dev

Prerequisites

  • Python 3.9+
  • Docker (optional, for containerized deployment)
  • Git

Your First Agent

Create and deploy your first AI agent with LunaOS. 

# Create a new agent
curl -X POST http://localhost:8001/api/agents \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-first-agent",
    "type": "chat",
    "config": {
      "model": "gpt-3.5-turbo",
      "temperature": 0.7,
      "system_prompt": "You are a helpful assistant."
    }
  }'

# Send a message
curl -X POST http://localhost:8001/api/agents/my-first-agent/messages \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello! How are you?",
    "context": {"user_id": "demo-user"}
  }'

Installing Plugins

Extend LunaOS functionality with plugins from our registry. 

# Install a plugin
curl -X POST http://localhost:8001/api/plugins/install \
  -H "Content-Type: application/json" \
  -d '{
    "name": "slack-integration",
    "version": "1.0.0"
  }'

# Configure the plugin
curl -X PUT http://localhost:8001/api/plugins/slack-integration/config \
  -H "Content-Type: application/json" \
  -d '{
    "webhook_url": "https://hooks.slack.com/your-webhook",
    "channel": "#general"
  }'

Deployment

Deploy LunaOS to production with Docker and cloud providers. 

# Docker deployment
docker-compose up -d

# Production deployment
make prod-setup
make deploy

# With Dapr (microservices)
make dapr-dev
